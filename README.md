# Horcrux RL

Ray RLlib을 활용한 강화학습 연구 프로젝트

## 프로젝트 구조

```
horcrux_rl/
├── notebooks/          # Jupyter 노트북 (실험 및 분석)
├── utils/              # 공통 유틸리티 함수
├── results/            # 실험 결과 저장
└── requirements.txt    # 패키지 의존성
```

## 설정

```bash
pip install -r requirements.txt
```

## 사용법

Jupyter 노트북을 실행하여 실험을 진행합니다:

```bash
jupyter notebook
```
